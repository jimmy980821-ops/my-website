01

細胞與胞器

對應掃描頁 1-7：細胞學說、細胞構造、細胞膜、胞器、原核與真核比較
概念細胞學說›
細胞是生物體構造與功能的基本單位。新的細胞來自原有細胞，細胞內含有遺傳物質，能把生命訊息傳給下一代。
人物重點
虎克 Hooke觀察軟木塞，提出「細胞」一詞。
雷文霍克 Leeuwenhoek觀察水中微生物與原生生物。
許旺、許萊登提出動物與植物都由細胞組成。
魏修 Virchow主張「細胞來自既有細胞」，補足細胞學說第三點。
構造動物細胞 vs 植物細胞›
動物細胞與植物細胞示意圖
構造動物細胞植物細胞
細胞膜有有
細胞壁無有，植物主要成分為纖維素；真菌為幾丁質、細菌為肽聚醣。
葉綠體無有，行光合作用。
液胞小且較不明顯大型液胞，儲存水分、養分與廢物。
中心粒常見多數高等植物沒有。
形狀較不固定因細胞壁較固定。
構造組成位置重點功能
細胞膜磷脂雙層、蛋白質、醣類、固醇類。包圍細胞外側；植物細胞位於細胞壁內側。區隔細胞內外水溶液，控制物質進出，並透過膜蛋白與醣類進行運輸、受器作用與細胞辨識。
細胞壁植物為纖維素；真菌為幾丁質；細菌為肽聚醣。細胞膜外側。保護細胞、維持細胞形狀並提供支持。
細胞核核膜、核質、染色體、核仁；染色體由 DNA 與蛋白質組成。真核細胞內。儲存遺傳物質、控制細胞活動；核仁製造核糖體組成成分。
細胞質胞質液、膜狀胞器與非膜構造；胞質液含水、離子、酵素與膠狀物質。細胞膜以內、細胞核以外。許多化學反應進行處，也是胞器分布與物質運輸的環境。
中心體 / 中心粒動物細胞常見，中心粒由微管蛋白構成。細胞核附近。細胞分裂時幫助形成紡錘絲。
核糖體RNA 與蛋白質組成。可游離於細胞質，也可附著在粗糙內質網、核膜外側；粒線體與葉綠體內也有。合成蛋白質。
整理細胞膜、細胞壁與胞器›
細胞膜：磷脂雙層與選擇性通透

細胞膜

主要由磷脂雙層、膜蛋白、醣類與固醇類組成。磷脂的親水端朝水溶液，疏水端相對，使膜具選擇性通透。

細胞壁

植物細胞壁主要為纖維素；真菌為幾丁質；細菌為肽聚醣。功能是支持、保護並維持細胞形狀。

細胞核

核膜為雙層膜並有核孔；核質含膠狀物質與酵素，是化學反應進行處；染色體由 DNA 與蛋白質組成。

細胞質

由胞質液、膜狀胞器與非膜構造組成。胞質液富含水、離子、酵素和膠狀物質，是細胞代謝反應的重要場所。
胞器位址型態功能
粗糙內質網 RER 細胞質中，核外側，並可與核膜相連。 單層膜囊狀管道，表面附有核糖體，所以外觀較粗糙。 合成、修飾並運輸蛋白質，特別是分泌蛋白與膜蛋白，也可作為物質運輸管道。
平滑內質網 SER 粗糙內質網外側或延伸處。 單層膜管狀構造，表面不附核糖體，所以較平滑。 合成脂質、固醇類，進行解毒並儲存鈣離子；肝細胞的 SER 特別發達。
高基氏體 細胞質中，通常位於內質網與細胞膜之間。 單層膜，由多層扁平囊堆疊而成，旁邊常有小囊泡。 接收內質網來的物質，進行修飾、分類、包裝，再形成囊泡分泌。
溶體 高基氏體外側形成，動物細胞的細胞質中較常見。 單層膜小囊，內含多種消化酵素。 胞內消化、胞器更新與分解外來物；也可協助汰換細胞。
液胞 細胞質中；植物細胞常有大型中央液胞。 單層膜囊狀構造，植物細胞的液胞通常很大。 儲存水分、養分、色素與廢物，並維持植物細胞膨壓。
粒線體 動植物細胞的細胞質中；耗能多的細胞通常較多。 雙層膜，內膜向內摺疊形成皺褶；含自己的環狀 DNA 與核糖體。 進行有氧呼吸，分解葡萄糖並產生大量 ATP。
葉綠體 植物綠色部位細胞的細胞質中，如葉肉細胞。 雙層膜，內含類囊體、基質、葉綠素、自己的 DNA 與核糖體。 進行光合作用，將光能轉成葡萄糖中的化學能。
中心體 / 中心粒 動物細胞核附近。 非膜構造，中心粒由微管蛋白組成。 細胞分裂時協助紡錘絲形成。
細胞骨架 細胞質中。 由微管、微絲與中間絲等蛋白質纖維構成。 維持細胞形狀、固定胞器位置，也和細胞運動與胞內運輸有關。

細胞與能量

對應掃描頁 8-10：ATP、光合作用、呼吸作用、能量轉換
能量ATP 與能量循環›
ATP 是細胞可直接使用的能量貨幣。ATP 分解成 ADP + Pi 時釋放能量；ADP 加上 Pi 則需要能量重新合成 ATP。
ATP 組成說明
腺嘌呤含氮鹼基，和核糖共同形成腺苷。
核糖五碳糖，連接腺嘌呤與磷酸基。
三個磷酸基ATP 名稱中的「三磷酸」來源。末端磷酸鍵水解時釋放能量，形成 ADP + Pi。
ATP - ADP 循環
反應意義
ATP + H2O → ADP + Pi + 能量放能反應，供應肌肉收縮、主動運輸、合成作用等。
ADP + Pi + 能量 → ATP儲能反應，能量多來自呼吸作用或光合作用。

分解代謝

大分子分解成小分子，通常釋放能量，例如呼吸作用分解葡萄糖產生 ATP。

合成代謝

小分子合成大分子，通常需要能量，例如光合作用把 CO2、H2O 合成葡萄糖。
作用光合作用›
光合作用把光能轉成化學能，儲存在葡萄糖中；主要在葉綠體進行。
光合作用流程圖：光反應 → 固碳反應
總反應6CO2 + 6H2O + 光能 → C6H12O6 + 6O2
意義把光能存入有機物，屬於合成作用；產物葡萄糖可供呼吸作用分解放能。
場所葉綠體。光反應在類囊體膜，固碳反應在基質。
光反應水分解產生 O2，電子與 H+ 使 NADP+ 還原成 NADPH，並把 ADP + Pi 合成 ATP。
暗反應 / 固碳反應固定 CO2，在基質中利用 ATP 與 NADPH 還原合成葡萄糖等醣類。
影響因子光強度、二氧化碳濃度、溫度、水分、葉綠素含量。
作用呼吸作用與發酵›
呼吸作用流程圖：葡萄糖逐步釋放能量
有氧呼吸總反應C6H12O6 + 6O2 → 6CO2 + 6H2O + ATP
糖解作用在細胞質中進行，葡萄糖分解成 2 個丙酮酸，淨產生 2 ATP。
有氧呼吸丙酮酸進入粒線體，在氧氣參與下產生 CO2、H2O 與大量 ATP，常記約 30-32 ATP。
酒精發酵酵母菌、部分植物根細胞缺氧時可進行，產生酒精與 CO2，ATP 少。
乳酸發酵乳酸菌、骨骼肌缺氧時可進行，產生乳酸，ATP 少。
影響因子溫度、氧氣濃度與酵素活性；溫度太高會使酵素失去活性。

光合作用

儲存能量：CO2 和 H2O 變成葡萄糖與 O2。

呼吸作用

釋放能量：葡萄糖和 O2 變成 CO2、H2O 與 ATP。

能量流動

能量流動是單向的：光能進入生態系，經生產者轉成化學能，最後多以熱能散失。

物質循環

元素可反覆循環，例如 CO2 在光合作用與呼吸作用之間轉換。

細胞分裂與遺傳

對應掃描頁 11-26：染色體、細胞週期、有絲分裂、減數分裂、孟德爾遺傳
分裂染色體與細胞週期›
DNA 複製前是一條染色質；複製後形成兩條姊妹染色分體，由著絲點連接。分裂時染色質濃縮成染色體，方便平均分配。
染色體構造：DNA、染色質、染色體
名詞重點
染色質平時呈細絲狀，主要由 DNA 與蛋白質組成。
染色體細胞分裂時染色質濃縮纏繞形成，較容易被平均分配。
基因DNA 上控制性狀的片段，可視為蛋白質或功能產物的訊息。
等位基因位於同源染色體相同位置、控制同一性狀的不同形式，例如 A 與 a。
同源染色體大小、形狀與基因位置相似，一條來自父方，一條來自母方。
姊妹染色分體DNA 複製後形成的兩條相同染色分體，由著絲點連接。
時期重點
間期 G1細胞生長，合成蛋白質與胞器，準備 DNA 複製。
間期 SDNA 複製，染色體由一條染色分體變成兩條姊妹染色分體。
間期 G2檢查 DNA、合成分裂所需物質，準備進入分裂期。
前期染色體凝縮，核膜、核仁逐漸消失，紡錘絲形成。
中期染色體排列在赤道板，紡錘絲連接著絲點。
後期姊妹染色分體分離，移向兩極。
末期染色體鬆開，核膜重建，接著細胞質分裂。

定律孟德爾遺傳›

分離律

成對的等位基因在形成配子時分離，每個配子只得到其中一個。

獨立分配律

不同性狀的等位基因在形成配子時可自由組合，前提是基因位於不同染色體或距離夠遠。
名詞說明
顯性在雜合狀態會表現出來的性狀，如 A。
隱性需同型合子才表現，如 aa。
基因型個體的基因組成，例如 AA、Aa、aa。
表現型實際呈現的性狀，例如高莖、矮莖。
同型合子成對等位基因相同，例如 AA 或 aa。
異型合子成對等位基因不同，例如 Aa。
親代 / 子代P 代表親代，F1 為第一子代，F2 為第二子代。
測交未知顯性個體 × 隱性同型合子，用來判斷未知個體基因型。
常見比例：單性雜交 Aa × Aa，基因型 1:2:1，表現型 3:1。測交 Aa × aa，表現型常見 1:1。雙性雜交 AaBb × AaBb，表現型常見 9:3:3:1。

基因型互動推算

輸入親代基因型，可使用單性雜交或雙性雜交，例如 Aa × Aa、AaBb × AaBb、AA × aa。
單性雜交：Aa × Aa
延伸非典型遺傳與血型›
類型重點例子
不完全顯性雜合子表現介於兩種同型合子之間。紅花 × 白花 → 粉紅花。
共顯性雜合子同時表現兩種等位基因的效果。AB 型同時有 A、B 抗原。
多基因遺傳一個性狀由多對基因控制，表現常呈連續分布。身高、膚色。
複等位基因族群中同一基因位置有超過兩種等位基因。ABO 血型。
性聯遺傳基因位於性染色體上，常見 X 聯遺傳。紅綠色盲、血友病。
血型可能基因型紅血球表面抗原血清抗體
A 型IAIA 或 IAiA 抗原抗 B
B 型IBIB 或 IBiB 抗原抗 A
AB 型IAIBA、B 抗原無抗 A、抗 B 抗體

血型補充重點
IA 與 IB彼此為共顯性，所以 IAIB 表現 AB 型。
i對 IA、IB 皆為隱性，所以 ii 才表現 O 型。
Rh 血型有 Rh 抗原為 Rh+，沒有為 Rh-；輸血時也需注意相容性。
性聯遺傳筆記重點：基因位於 X 或 Y 染色體上。男性為 XY，只有一條 X，所以 X 聯隱性性狀較容易在男性表現；女性 XX 通常需兩個隱性等位基因才表現。
項目重點整理
性染色體女性 XX、男性 XY；卵細胞只帶 X，精子可帶 X 或 Y，因此父方精子決定子女性別。
X 聯隱性如紅綠色盲、血友病；男性只要 X 上帶隱性基因就會表現。
帶因者女性雜合通常不表現，但可把隱性基因傳給子代。

DNA、蛋白質與基因科技

對應掃描頁 27-31：核酸、DNA 複製、轉錄、翻譯、基因工程
分子核酸與 DNA›
DNA 與 RNA 結構比較
核酸基本單位核酸由許多核苷酸連接而成；每個核苷酸包含一個磷酸基、一個五碳糖與一個含氮鹼基。
項目DNARNA
五碳糖去氧核糖核糖
鹼基A、T、C、GA、U、C、G
構造雙股螺旋多為單股
功能儲存遺傳訊息，可作為複製與轉錄模板。參與蛋白質合成，可分為 mRNA、tRNA、rRNA 等。
所在位置主要在細胞核；粒線體與葉綠體也有自己的 DNA。可在細胞核、細胞質與核糖體中發現。
DNA 鹼基配對：A 配 T，C 配 G。複製時兩股分開，各自作為模板合成互補新股。
DNA 複製：半保留複製
DNA 複製重點說明
半保留複製每個子代 DNA 都含一股舊股與一股新股。
解旋雙股 DNA 先分開，露出兩股模板。
互補配對A 對 T、C 對 G，新股依模板合成。
意義讓細胞分裂前能複製遺傳訊息，使子細胞取得完整 DNA。
中心法則轉錄與翻譯›
中心法則：DNA → RNA → 蛋白質
DNA：儲存蛋白質設計圖。
轉錄：DNA 的部分訊息被複製成 mRNA。
mRNA：把訊息帶到核糖體。
翻譯：核糖體讀取 mRNA 密碼子，tRNA 帶來胺基酸。
蛋白質：胺基酸依序連接，形成具有功能的蛋白質。
名詞重點
密碼子mRNA 上每三個鹼基為一組，決定一個胺基酸。
反密碼子tRNA 上與密碼子互補配對的三個鹼基。
核糖體翻譯作用場所，讓 mRNA 與 tRNA 配對並形成多胜肽。
mRNA信使 RNA，將 DNA 訊息帶到核糖體。
tRNA轉運 RNA，攜帶特定胺基酸，反密碼子與 mRNA 密碼子配對。
rRNA核糖體 RNA，與蛋白質共同組成核糖體。
起始密碼AUG 通常作為起始密碼，對應甲硫胺酸。
終止密碼UAA、UAG、UGA 不對應胺基酸，使翻譯停止。
科技基因工程與應用›
基因工程操作流程：限制酶、質體、重組 DNA
步驟說明
取得目標基因用限制酶切割 DNA，取得需要的基因片段。
接入載體常用質體作為載體，將目標基因接入。
轉入宿主把重組 DNA 送入細菌或其他細胞。
篩選與培養找出成功轉殖的細胞，讓其大量繁殖或表現蛋白質。
工具 / 名詞功能
限制酶像分子剪刀，可在特定序列切割 DNA。
DNA 接合酶像分子膠水，可把 DNA 片段接合成重組 DNA。
質體細菌中的小型環狀 DNA，常作為基因載體。
重組 DNA把不同來源的 DNA 片段接在一起形成的新 DNA。
轉殖生物含有外來基因並能表現該基因的生物。

演化與生物分類

對應掃描頁 32-39：演化理論、分類系統、演化證據、親緣關係樹
理論演化思想›
拉馬克 vs 達爾文：長頸鹿例子
人物 / 理論重點問題或意義
拉馬克用進廢退、獲得性遺傳。認為常使用的構造逐漸發達，不常使用則退化；但獲得性狀通常不能遺傳。
達爾文天擇說：個體有差異，環境選擇適合者留下後代。演化不是個體改變，而是族群中有利性狀比例改變。
現代演化理論結合天擇、遺傳學、突變、基因重組、族群概念。以族群基因頻率改變解釋演化。
魏斯曼實驗連續剪掉老鼠尾巴，後代仍有尾巴。說明體細胞後天變化不會直接遺傳，遺傳主要與生殖細胞基因有關。
自然選擇的重點：先有變異，再由環境篩選。不是生物「想要變」才變。
觀念整理
用進廢退拉馬克提出，生物特徵不是永久不變，越常使用會越發達，不常使用逐漸退化。
獲得性遺傳拉馬克認為生物一生因環境或行為產生的性狀可傳給下一代。
天擇達爾文認為族群原本就有個體差異，環境讓適合者生存、繁殖較多。
生存競爭生物繁殖快但資源有限，因此會產生競爭，自然選擇在競爭中發生。
證據演化證據›
演化證據：化石地層、同源器官、分子序列
證據說明
化石證據保存在地層中的生物遺體或生活痕跡。越古老地層的化石通常越簡單。
比較解剖同源器官顯示共同祖先，如人手、蝙蝠翼、鯨前肢。
痕跡構造祖先具有功能、經長時間演化後在現今物種失去或弱化功能的構造，如人類尾椎。
同功構造功能相似但來源不同，常因相似環境而演化出相似功能，不一定代表親緣近。
胚胎發生不同脊椎動物早期胚胎相似，顯示親緣關係。
生物地理地理隔離會造成不同地區物種分化。
分子證據DNA 或蛋白質序列越相似，親緣關係通常越近。
同源構造看「來源」相同；同功構造看「功能」相同。判斷親緣關係時，同源構造與分子序列通常比外表像不像更可靠。
分類生物分類與親緣關係›
親緣關係樹：越晚分支，關係越近
分類階層界 → 門 → 綱 → 目 → 科 → 屬 → 種
種能自然交配並產生具有生殖能力後代的一群生物。
雙名法屬名 + 種小名，用拉丁文表示。
五界原核生物界、原生生物界、真菌界、植物界、動物界。
三域細菌域、古菌域、真核生物域。
五界細胞型態營養方式與例子
原核生物界原核細胞，無核膜。細菌、藍綠菌；有些自營，有些異營。
原生生物界真核，多為單細胞或簡單多細胞。草履蟲、眼蟲、藻類等。
真菌界真核，有細胞壁，主要成分為幾丁質。異營，靠吸收養分；如黴菌、酵母菌、菇類。
植物界真核，多細胞，有纖維素細胞壁與葉綠體。多為自營，行光合作用。
動物界真核，多細胞，無細胞壁。異營，通常可運動並攝食。
親緣關係樹中，分支點代表共同祖先；越晚分開，親緣關係越近。
親緣樹判讀說明
分支點代表共同祖先。
越晚分支通常親緣關係越近。
最早分支代表較早和其他類群分開，不代表比較低等。
判讀重點看共同祖先遠近，不只看圖上左右距離。
整理生物多樣性與演化比較›
比較拉馬克達爾文
變異來源環境使個體產生需求。族群原本就有差異。
演化方式用進廢退、獲得性狀遺傳。自然選擇使適合環境者留下較多後代。
例子長頸鹿因常伸長脖子而變長。原本脖子較長者較容易取得食物，留下較多後代。

細胞學說
生物由細胞組成；細胞是構造與功能基本單位；新細胞來自既有細胞。
磷脂雙層
細胞膜主要構造，親水端朝外，疏水端朝內，使膜具選擇性通透。
細胞壁
植物為纖維素，真菌為幾丁質，細菌為肽聚醣；功能是保護、支持並維持形狀。
細胞質
由胞質液、膜狀胞器和非膜構造組成；胞質液含水、離子、酵素與膠狀物質。
粒線體
有氧呼吸主要場所，將葡萄糖能量轉為 ATP。
葉綠體
光合作用場所，含葉綠素，可把光能轉成糖類中的化學能。
ATP
由腺嘌呤、核糖與三個磷酸基組成；水解成 ADP + Pi 時釋放能量。
光合作用
6CO2 + 6H2O + 光能 → C6H12O6 + 6O2。
呼吸作用
C6H12O6 + 6O2 → 6CO2 + 6H2O + ATP。
有絲分裂
產生兩個與母細胞相同的子細胞，用於生長、修補與無性生殖。
減數分裂
產生四個染色體數減半且彼此不同的配子。
同源染色體
一條來自父方、一條來自母方，大小形狀與基因位置相似。
分離律
成對等位基因在形成配子時分離，每個配子只得到一個。
獨立分配律
控制不同性狀的基因在形成配子時可自由組合。
ABO 血型
複等位基因例子：IA、IB、i；IA 與 IB 共顯性，i 為隱性。
DNA 複製
以舊股為模板合成新股，A-T、C-G 互補配對。
核苷酸
核酸基本單位，由磷酸基、五碳糖與含氮鹼基組成。
中心法則
DNA 經轉錄形成 RNA，RNA 經翻譯形成蛋白質。
