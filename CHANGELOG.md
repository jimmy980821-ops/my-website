# 🚀 Website Change Log

所有此專案的重要更新都會記錄於此。

## 📌 v1.4.0 - 2026-07-09

### ✨ Added

新增「學測倒數」Dashboard 頁面，包含距離學測剩餘天數、今日學習目標、最近完成項目與學習進度。

新增「學測 7000 隨機單字」功能，讓每日英文學習可以隨機抽取單字練習。

新增「暑假學習計畫表」與今日讀書計畫功能，方便整理每日讀書進度。

新增作品頁分類，包含專題研究、程式作品、設計作品與學習成果。

新增「高中學習歷程資料庫」分類頁，整理國文、英文、數學、自然、社會、探究實作、課程成果與學習反思。

新增資料庫檔案管理功能，支援上傳、重新命名、刪除與分類整理。

新增學習歷程 PDF 檔案，並整合至資料庫中方便瀏覽。

### 🎨 Improved

將網站定位從單純作品集調整為 Learning Hub / Digital Knowledge Garden，讓網站能同時展示作品、學習筆記、學校成果與個人研究。

修改首頁 Hero 文案，改為更符合學習中心定位的「Learning Never Stops.」。

調整首頁 CTA 按鈕為「學測倒數」、「瀏覽作品」與「學習資料庫」。

修改首頁功能卡片，改為介紹學測準備、作品、高中學習歷程資料庫與關於我。

重新整理 Navigation Bar 順序：首頁、學測倒數、作品、高中學習歷程資料庫、關於我。

優化 About 頁內容，不再以履歷方式呈現，改為自我介紹、建立網站的原因、學習理念與未來目標。

改善淺色模式下搜尋欄、輸入框與選單文字可讀性。

修正多處中文顯示亂碼問題，提升整體閱讀體驗。

### 🔧 Fixed

修正作品查看詳情按鈕跳轉錯誤的問題。

修正資料庫作品分類放置相反的問題。

修正上傳檔案重新整理或關閉網頁後容易消失的問題。

修正資料庫檔案開啟時出現檔案不存在的問題。

修正部分按鈕無法點擊與狀態文字顯示異常的問題。

修正首頁標題與內容在不同螢幕尺寸下重疊的問題。

### 🚀 Optimized

移除內嵌 Base64 PDF，將 PDF 改為獨立靜態檔案，降低首頁檔案大小並提升載入速度。

拆分網站檔案結構，將 HTML、CSS、JavaScript 與 PDF 資源分開管理。

改善 GitHub Pages 部署檔案結構，讓 PDF 與 JavaScript 檔案可以正常開啟。

優化資料庫搜尋與分類篩選體驗，讓使用者可以依科目與年級快速找到資料。

提升網站在手機、平板與桌面裝置上的穩定性。
